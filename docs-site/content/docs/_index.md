Welcome to the Neuracore documentation! Neuracore is a powerful robot learning library that enables data collection and visualization, model training, deployment, and real-time inference with support for custom data types.

## Quick Start

Get started with Neuracore in minutes:

```bash
pip install neuracore
```

For training and ML development:
```bash
pip install neuracore[ml]
```

For bulk importing datasets:
```bash
pip install neuracore[import]
```
